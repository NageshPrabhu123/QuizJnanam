# QuizJnanam — AI Proctored Online Exam Platform

QuizJnanam is a free AI-powered proctored online exam platform 
built for colleges and universities in India.

## 🔗 Live Demo
👉 https://quizjnanam.netlify.app

## Features
- AI face detection monitoring
- Real-time violation detection
- Camera + screen share proctoring
- Teacher dashboard with analytics
- Instant results and PDF reports
- MCQ, Descriptive, True/False questions
